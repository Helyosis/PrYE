conclusion()[source]¶

Renvoie None si les prémisses conduisent à une contradiction, ou un dictionnaire associant à chaque variable “A”, “B”, “C” et “D” soit True si elle est prouvée, False si la négation est prouvée, on None si on ne peut rien conclure.

Renvoie True si l’insertion est possible, False sinon.

Enlève la carte en position index de la prémisse premise à condition qu’elle vienne d’être ajoutée. Renvoie la carte en question ou None si on ne peut pas l’enlever.
